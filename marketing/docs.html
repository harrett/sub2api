Codex 简介
OpenAI Codex CLI 是 OpenAI 官方推出的命令行 AI 编程助手，类似于 Claude Code，但使用 OpenAI 的模型（如 GPT-4o、o1、o3 等）。通过 Sub2API 接入，无需海外信用卡，直接用人民币充值调用。
推荐使用 WSL 或 Git Bash
Codex 在 Windows 上推荐使用 WSL（Windows Subsystem for Linux）安装和运行，兼容性最好。如果不想装 WSL，至少使用 Git Bash 来运行 Codex。PowerShell 兼容性较差，可能遇到文件锁定、写入失败（error write file）等问题。

2配置环境变量

按 Win+X → 选择「终端管理员」，运行（替换为你的域名和 Key）：

POWERSHELL · 管理员
[System.Environment]::SetEnvironmentVariable("OPENAI_BASE_URL", "https://api.your-domain.com/v1", [System.EnvironmentVariableTarget]::Machine)
[System.Environment]::SetEnvironmentVariable("OPENAI_API_KEY", "sk-sub2-YOUR_KEY", [System.EnvironmentVariableTarget]::Machine)

设置后需重新打开终端才能生效。

验证
POWERSHELL

之前用过其他中转？清理旧配置
POWERSHELL · 管理员
# 检查现有配置

# 清理
[Environment]::SetEnvironmentVariable('OPENAI_BASE_URL', $null, 'Machine')
[Environment]::SetEnvironmentVariable('OPENAI_API_KEY',  $null, 'Machine')

0前置要求：安装 Git

Claude Code 依赖 Git 运行。若启动后看到以下错误，说明需要先安装 Git：

ERROR
Claude Code on Windows requires git-bash (https://git-scm.com/downloads/win).
If installed but not in PATH, set environment variable:
  CLAUDE_CODE_GIT_BASH_PATH=C:\Program Files\Git\bin\bash.exe

前往 git-scm.com 下载安装，保持默认选项。安装后重启终端验证：

POWERSHELL
git --version
Git 已安装但提示找不到 bash？自动检测路径
POWERSHELL · 管理员
$p = @(
  "$env:ProgramFiles\Git\bin\bash.exe",
  "${env:ProgramFiles(x86)}\Git\bin\bash.exe",
  "$env:LOCALAPPDATA\Programs\Git\bin\bash.exe"
)
$f = $p | Where-Object { Test-Path $_ } | Select-Object -First 1
if ($f) {
    [Environment]::SetEnvironmentVariable("CLAUDE_CODE_GIT_BASH_PATH", $f, "Machine")
    "已设置: $f"
} else { "未找到 Git，请先安装" }
